01 — Abstract

Intelligent Diagnostics for Preventive Healthcare

Traditional diagnosis relies on lab tests and in-person appointments — allowing diseases to progress undetected. This AI system uses ensemble ML models trained on real patient symptom data to predict diseases at the earliest stage.

Drawing upon a Kaggle dataset of 4,920 records with 132 symptom features across 41 disease classes, our ensemble approach outperforms any single classifier by up to 18 percentage points.

AI HealthcareEarly Detection NLP ProcessingEnsemble Models Risk StratificationPreventive Medicine
Problem Statement
How can scalable, real-time AI systems predict diseases from symptom descriptions, enabling preventive care at the earliest stage across diverse healthcare settings?
Dataset & Scope
Kaggle Disease-Symptom Dataset: 4,920 patient records, 132 binary symptom features, 41 disease classes. Trained with stratified 80/20 split and 10-fold cross-validation.
Key Contribution
Ensemble majority-vote fusion of Random Forest (94%), Naive Bayes (77%), and Decision Tree (89%) achieves 96% accuracy — an 18-point gain over the weakest individual model.
02 — Core System

Three Pillars of Intelligent Diagnosis

The system is architected around three deeply integrated modules that work in sequence from symptom capture to clinical action.

01
🔬
Input Layer
Symptom Input Engine

Multi-modal input via structured forms or natural language text. Users describe symptoms in plain language — the AI preprocesses, normalizes and encodes them into 132-dimensional feature vectors. Supports English and regional language inputs for rural healthcare access.

02
🧠
ML Core
ML Classification Model

Ensemble of Random Forest, Naive Bayes & Decision Tree classifiers trained on 4,920 patient records across 41 disease classes. Majority-vote fusion achieves >95% accuracy — outperforming any single classifier by up to 18 percentage points.

03
🚨
Output Layer
Risk Stratification

Outputs ranked disease predictions with confidence scores and severity tier — Low, Medium, or Critical. Provides actionable next-step guidance: home monitoring, specialist referral, or immediate emergency escalation as clinically appropriate.
